## Run local server:

Open PowerShell and run:

```powershell
cd C:\Users\joant\Desktop\infinite-tower\application\infinite-tower.github.io
```

---

## Open in browser:

**PC:**
```
http://localhost:8000
```

**Mobile (same WiFi):**
```
1. In PowerShell: ipconfig
2. Find your IPv4 (e.g., 192.168.1.25)
3. On mobile: http://192.168.1.25:8000
```

---

## Test mobile view on PC:

1. Open Chrome
2. Press **F12** (DevTools)
3. Click **Toggle Device Toolbar** (phone icon) or **Ctrl+Shift+M**
4. Select device (iPhone, Pixel, etc.)
5. Test portrait and landscape

---

## Stop server:

Press **Ctrl+C** in PowerShell
